run(key, onInitopt) → {Promise.<any>}

Description:

  • GraftSuspend
    프로세스 제어자를 등록 및 실행 대기 합니다. 프로세스의 등록 및 실행은 프로세스 제어자가 반환(귀결: GraftSuspend.resolve / 거부: GraftSuspend.reject)되기 전까지 재등록 할 수 없습니다.

Example
// 코루틴 등록 및 프로세스 대기
graft.suspend.run('xxx')
     .then(receive=> {
         // 코루틴 수행 귀결 처리 결과를 반환 받습니다.
         graft.log.out('정상', receive);
     })
     .catch(reason=> {
         // 코루틴 수행 거부 처리 결과를 반환 받습니다.
         graft.log.out('오류', reason);
     });
