브릿지

웹과 네이티브의 기능 요청에 대한 메시지를 주고 받는 역할을 합니다. 웹에서는 graft.bridge.postMessage 메소드를 이용하여 네이티브에 기능 수행에 대한 요청 메시지를 전달 합니다. 네이티브에서는 Graft.postMessage 기능을 호출 하여 기능 수행에 대한 결과에 대한 응답 메시지를 전달 합니다. 응답에 대한 메시지는 웹에서 요청한 메시지의 trid값을 포함하여 전달 합니다.

메시지 구조

{
    // 메시지 거래를 위해 채번된 ID
    "trid": "w112233",
    // 기능 요청 커맨드
    "command": "",
    // 기능 요청 시 전달값
    "payload": {},
    // 기능 수행 후 응답값
    "receive": {}
}
  • trid: 메시지 거래를 위해 채번된 ID값으로
