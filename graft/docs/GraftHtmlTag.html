Type Definitions

tag(…args) → {GraftElement}

Description:

  • GraftHtmlTag
    태그 이름에 해당 하는 엘리먼트를 생성하며, 전달 받은 매개변수의 타입에 별 다음과 같은 기능을 합니다.
    string | number: 엘리먼트의 HTML 마크업을 랜더링 합니다.
    function: 엘리먼트 생성 초기화 함수 입니다.
    object: 엘리먼트 속성을 세팅 합니다.
    array: 엘리먼트에 서브 엘리먼트를 추가 합니다.

Example
// string | number: 엘리먼트의 HTML 마크업을 랜더링 합니다.
// 생성된 HTML: <div><strong>text</strong></div>
const div= graft.el.div('<strong>text</strong>');

// function: 엘리먼트 생성 초기화 함수 입니다.
// 생성된 HTML: <div>xxx</div>
const div= graft.el.div(_it=> {
    _it.lot.text= 'xxx';
});

// object: 엘리먼트 속성을 세팅 합니다. 
// 생성된 HTML: <div id="xx1" data-bind="xx2"></div>
const div= graft.el.div({id:'xx1', dataBind:'xx2'});

// array: 엘리먼트에 서브 엘리먼트를 추가 합니다.
// 생성된 HTML: <ul><li>xxx</li><li>yyy</li></ul>
const ul= graft.el.ul([
    graft.el.li('xxx'),
    graft.el.li('yyy'),
]);
